# package_chatbot
Gọi hàm này để tích hợp vào app.
Param cần truyền: context, urlAPI, urlChatBot , userName, fullName và soDienThoai.


example:
 ExtensionFunction.instance.goChatBot(
     context: context,
     urlAPI: 'urlAPI',
     urlChatBot: 'urlChatBot',
     userName: 'userName',
     fullName: 'fullName',
     soDienThoai: 'soDienThoai');
